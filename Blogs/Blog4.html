IxD process

Interaction Design also known as IxD refers to or centers on the overall design process on how a user interacts with a digital system by creating a clear way to represent its information, actions, and visuals. I wish to implement these same principles in my website. the design process below provides a clear way to do this step by step. Its design process goes as follows:

1. Goal Alignment

My goal for this project is to create a portfolio website that I can use to apply for future jobs and showcase my vast skill set and coding capabilities. I want to be able to update and improve this website whenever I please, ensuring it remains relevant for future employers. I also want users to gain a deep understanding of who I am, making me a desirable candidate to hire.

2. Information structure

3. User Flow

For my user flow, I want to create an application that is easy and simple to navigate. Users should find clear instructions and bold, readable text that guides them to the specific section of the website they wish to visit. Additionally, visuals and imagery will provide a brief summary of each section, reinforcing the key points of any text it goes with.

4. Interface elements with my updated wireframes

Navigation is my main priority here but i wish for everything to be portrayed in a professional format. Easy to use for anyone.

5. Design (possible style guides and general inspirations)

Provides simple navigation, bold texts providing clear titles, Font used for different sections( if paragraph smaller font else bigger font for titles and subheadings) Easy colours(easy on the eyes).

Visuals and imagery are clear and concise and appealing to the user, the top provides a way for navigation without the standard way navigation bar. Provides basic colour scheme which will be easy to implement.
